Finding cycles/circuits/components in a graph    
The c++ files is a class that takes a graph (from console input) in the linked list form and performs a depth first
search, it finds the number of connected components and the first cycle in the graph.
The java file is a class that takes a graph (no main()- array[][] passed as parameter) in the 2D matrix form and finds if
it has an Eulerean circuit and/or an Eulerean path.
